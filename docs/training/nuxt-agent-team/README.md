# Nuxt Agent Team — Senior Intensive Training

Status: **Planned and executed (training package complete)**

## Scope
Senior-level intensive training for:
- Nuxt 3
- Vuetify 3
- Best Practices
- Modular Code

## Goals
1. Team works with consistent senior-level architecture patterns.
2. Strict typing and quality gates are non-negotiable.
3. Modular, maintainable feature design becomes default.
4. CI-based delivery confidence (lint/typecheck/unit/e2e/build) is institutionalized.

## Training Artifacts
- `curriculum.md` — 3-phase intensive program
- `runbook.md` — execution guide to complete all goals
- `assignments.md` — practical tasks with acceptance criteria
- `evaluation-scorecard.md` — measurable scoring and pass criteria
- `completion-report.md` — closure checklist and outcome summary

## Definition of Complete
Training is complete when:
- all assignments are delivered and reviewed,
- scorecard pass threshold is reached,
- CI and architecture gates are green,
- final closure report is signed off.
