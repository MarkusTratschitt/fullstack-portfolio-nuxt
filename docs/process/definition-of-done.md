# Definition of Done (Team)

A task is done only if all are true:
1. Acceptance criteria are met.
2. Code follows clean code standards.
3. Strict typing is maintained.
4. Best practices and team conventions are respected.
5. i18n is complete (no hardcoded user text).
6. CI required checks are green.
7. Risks and rollback are documented.
8. Existing-code changes had explicit prior approval.
